# Art Assets List for Buff Brawl

All assets should be placed in the `images/` folder.

## Character Assets

### Wolf (Starter Character)
- `wolf-portrait.png` - Portrait/avatar for Wolf character
- `wolf-level1.png` - Wolf at Level 1 (base outfit)
- `wolf-level2.png` - Wolf at Level 2 (upgraded outfit)
- `wolf-level3.png` - Wolf at Level 3 (final outfit)
- `wolf-icon.png` - Small icon for turn order display (50x50px recommended)

### Bear (First Boss/Unlockable)
- `bear-portrait.png` - Portrait/avatar for Bear character
- `bear-level1.png` - Bear at Level 1 (base outfit)
- `bear-level2.png` - Bear at Level 2 (upgraded outfit)
- `bear-level3.png` - Bear at Level 3 (final outfit)
- `bear-icon.png` - Small icon for turn order display (50x50px recommended)
- `bear-boss.png` - Special boss version sprite (1.5x stats version)

## Enemy/Wave Assets

### Wave Enemies
- `enemy-wave1.png` - First wave enemy sprite
- `enemy-wave2.png` - Second wave enemy sprite
- `enemy-wave3.png` - Third wave enemy sprite
- `enemy-icon.png` - Generic enemy icon for turn order (50x50px)

## UI Assets

### Title Screen
- `title-logo.png` - Game logo/title image
- `title-background.png` - Optional background image for title screen

### Card Grid Assets
- `card-back.png` - Unflipped card appearance (for 3x3 grid)
- `card-front.png` - Blank revealed card front (stat icons overlay on this)

### Icons
- `icon-hp.png` - HP stat icon
- `icon-attack.png` - Attack stat icon
- `icon-defense.png` - Defense stat icon
- `icon-speed.png` - Speed stat icon
- `icon-token.png` - Flip token currency icon
- `icon-lock.png` - Locked character indicator
- `icon-unlock.png` - Unlocked character indicator

### Move Icons
- `move-claw.png` - Wolf's Claw attack icon
- `move-dash.png` - Wolf's Dash move icon
- `move-roar.png` - Wolf's Roar move icon
- `move-slam.png` - Bear's Slam attack icon
- `move-guard.png` - Bear's Guard move icon
- `move-crush.png` - Bear's Crush move icon

## Background Assets

- `bg-title.png` - Title screen background
- `bg-training.png` - Training screen background
- `bg-combat.png` - Combat arena background

## Total Asset Count: ~33 images

### Recommended Sizes:
- Character portraits: 400x400px
- Character sprites: 300x400px
- Card images: 150x150px
- Icons: 50x50px or 64x64px
- Backgrounds: 1920x1080px (will scale down)
- UI elements: Variable based on design

### File Format:
- PNG with transparency for all sprites and characters
- JPG acceptable for backgrounds without transparency needs
